Setup
1) Ensure backend is running at http://127.0.0.1:8000
2) Install Node.js (v18+ recommended)
3) Install deps: npm i
4) Run dev: npm run dev

Config
- To point to a different API, create a .env file:
  VITE_API_BASE_URL=http://127.0.0.1:8000

Routes
- /signin, /signup
- /slots (customer)
- /bookings (customer)
- /admin, /admin/slots (admin)
